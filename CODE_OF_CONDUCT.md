# Code of Conduct

We are committed to a welcoming, inclusive, and respectful community.

- Be kind, constructive, and professional.
- No harassment, discrimination, or hateful speech.
- Assume good intent; focus on ideas, not individuals.
- Respect privacy and confidentiality (no secrets in code or issues).

This project follows the spirit of the Contributor Covenant. For more details, see https://www.contributor-covenant.org.

If you experience or witness unacceptable behavior, please report it to the maintainers via the contact channels in `SECURITY.md`.
